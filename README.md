# API Integration Demo

## Getting Started

```bash
# Clone the repository
git clone https://github.com/ethan-rng/wfn-demo.git
cd wfn-demo
git checkout -b demo/empty-template origin/demo/empty-template

# Set up environment variables
# MacOS/Linux:
cp .env.example .env
# Windows:
copy .env.example .env

# Install dependencies and start
npm install
npm start
```

## Lesson Topics

1. Request Structure (Body, Method Type, Headers)
2. Fetch and Async/Await
3. Environment Variables
4. SDKs

## Instructions
Check out the `demo/template` branch for code templates and examples. Return to `main` branch using GitHub to see the completed solutions in `App.js`.
