# API Integration Demo

## Getting Started
```bash
git clone https://github.com/ethan-rng/wfn-demo/tree/demo/empty-template
npm install
npm run start
```

## Lesson Topics

1. Request Structure (Body, Method Type, Headers)
2. Fetch and Async/Await
3. Environment Variables
4. SDKs

## Instructions
Check out the `demo/template` branch for code templates and examples. Return to `main` branch using GitHub to see the completed solutions in `App.js`.
